# Beegood IT Invoicing Core Module
Provides invoicing for subscribable resources.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "bgit-invoicing"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install bgit-invoicing
```
